This project is a responsive web page developed as part of a Web Development Assignment.

The page is built using vanilla HTML, CSS, and JavaScript without using any frameworks or libraries.

Features:
* Responsive design for desktop, tablet, and mobile devices
* Sticky header functionality while scrolling
* Interactive image carousel
* Zoom preview effect on hover
* Smooth transitions and animations

Technologies Used:
* HTML5
* CSS3 (Flexbox / Grid)
* JavaScript
